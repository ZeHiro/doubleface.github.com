Konnectors : qu'est ce que c'est ?

  • Récupère vos données sur différents services web
  • Les sauve dans votre cozy (bien rangés)
  • Un projet très communautaire

Examples

  • Vos factures de téléphone ou de box internet (sfr, red, bouygues, orange, sosh, free)
  • Vos remboursements ameli
  • Vos factures et événements SNCF
  • Vos podcasts
  • Vos twitts
  • Et même vos commits github (pour les devs dans la salle)

Idéal pour commencer à contribuer

Un connecteur est simplement un petit script javascript

Il y a de nombreux exemples et un super guide pour commencer à s'y mettre

https://github.com/cozy-labs/konnectors/wiki
