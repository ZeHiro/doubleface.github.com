Konnectors : qu'est ce que c'est ?

  • Récupère vos données sur différents services web
  • Les mets en forme de manière uniforme
  • Les sauve dans votre cozy (bien rangés)
  • Un projet très communautaire

Un connecteur ?

Un simple script javascript

Deux types de connecteurs

  • Ceux utilisant un API json
  • Ceux scrappant un site web

Scrapper ?

Simuler les actions utilisateur dans le navigateur

  • Saisie de formulaires
  • Appui bouton

Simuler les actions DU navigateur

  • Envoyer des requètes HTTP
  • Télécharger des fichiers

Étapes d'un connecteur

  1. Log in
  2. Parser la page où sont les données
  3. Remettre en forme les données
  4. Sauvegarder les données

Outils

Un navigateur !

  • Inspecteur de DOM (Ctrl + Shift + C sur Firefox)
  • Analyseur réseau (Ctrl + Shift + Q sur Firefox)
  • Sources de la page (Ctrl + U)

Ouvrir les outils de dev avant d'aller sur le site du provider
